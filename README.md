# AI News: Three Surfers

*Sniffed it out for you.*

追いきれないAIニュースを、3人のキャスターが波乗りする。

---

AI業界の変化は速い。モデル競争だけでなく、エージェント化・業務自動化・SaaS組み込みと、仕事の現場に直接影響する動きが毎日起きている。

このアプリは、そのすべてを追いきれない人のために作った。3人のAIキャスターがニュースの波を拾い、会話できる話題に変える。単なるニュース一覧ではなく、**キャスターと対話できるAIニュース番組**として設計されている。

---

## 3人のキャスター

**Echo 🦭 — 今日のAI地図**
AI業界全体の動きを地図として整理する。大手企業の発表、モデル更新、規制・著作権・社会影響、日本国内のAI政策や企業動向。毎朝「今日のAI界隈、どこが動いたか」を把握するためのチャンネル。

**Lucky 🦝 — 現場のAI**
「実際どう使われているか」「仕事にどう効くか」を現場視点で拾う。AIエージェントの実利用、ローコード/ノーコード、RPA連携、社内AI、議事録・資料作成・問い合わせ対応。業務に直結するニュースを集める。

**Sonar 🐬 — エージェント戦線**
AIが仕事の流れを変え始めている最前線を観測する。AIエージェント、MCP/Skills/tools/API連携、SaaSへのAI組み込み、ブラウザ操作AI、AIによる開発・保守・テスト。「次に仕事が変わる場所」を先読みするためのチャンネル。

---

## 機能

ニュース取得はチャンネルごとに最適化された検索クエリでリアルタイムに行われる。表示・切替のたびに最新情報を取りに行く。

- **今日の3行まとめ** — チャンネルごとに、その日の動向を3行に凝縮
- **キャスターのひとこと** — そのキャスター固有の視点で今日を一言で表現
- **ニュースカード** — タイトル・なぜ重要か・タグ・情報源の4点セット（5件）
- **ニュース解説** — カードをタップするとキャスターが「そもそも何？」から解説
- **深掘りチャット** — キャスターとリアルタイムで対話。以下の切り口で話せる
  - かんたんに説明して
  - 仕事に関係ある？
  - 学院で話すなら？
  - ちょっと批判的に見ると？
  - この流れ、今後どうなる？
- **🌍ワールド / 🇯🇵日本 切替** — リージョンで検索クエリが切り替わり、即再取得

---

## 使い方

1. [Anthropic Console](https://console.anthropic.com/) でAPIキーを取得・チャージ
2. アプリ上部の入力欄にAPIキーを入れて「起動」
3. チャンネルを選ぶ（地図 / 現場 / 戦線）
4. ニュースカードをタップして解説を聞く
5. 深掘りチャットモードでキャスターと話す

## 取材ポリシー

情報源は英語・米国発が7〜8割（TechCrunch・Wired・Bloomberg等が中心）。🇯🇵日本モードでは「日本・国内」クエリを追加して優先するが、海外発が混ざることはある。AIが要約・整理するため原文とニュアンスが異なる場合がある。感覚的な情報収集・話題のきっかけとして使うことを想定している。
